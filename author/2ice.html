My Vim Config

By 2ice

In vim.

tags: vim 

set nu   "显示行号
set nocompatible   "如果没有这个，配置中<CR>等无法被识别
set shiftwidth=4   "自动缩进所使用的空白符长度
set tabstop=4   "tab的空格长度
set autoindent   "新增一行时，把当前行的indent应用到新行
set cursorline   "当前行加下划线
set hlsearch   "搜索时高亮
set ignorecase   "搜索时忽略大小写
syntax on   "开启语法高亮

"设置按<F7>为取消搜索高亮
nmap <silent> <F7> :nohl<CR>
cmap <silent> <F7> :nohl<CR>

"显示或隐藏NERDTree区域
nmap <silent> <F3> :NERDTreeToggle<CR>
cmap <silent> <F3> :NERDTreeToggle<CR>

"重新打开一个文件时，跳转到上次打开最后的位置
if has("autocmd")
  au BufReadPost * if line("'\"") > 1 && line("'\"") <= line("$") | exe "normal! g'\"" | endif
  endif

其中：

NERDTree是vim的一个树形目录插件。可以在vim中浏览目录。

map是一个vim的映射命令，将常用的很长的命令映射到一个新的功能键上。
